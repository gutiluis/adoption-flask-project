# Adoption Flask Project

A simple **Flask web application** for managing pet adoptions.  
This project allows users to view available pets, create profiles, and submit adoption requests.

---

## Features

- View all pets available for adoption
- View pet details and profiles
- User login and registration (optional)
- Secure input handling (prevents XSS)
- Flask-based backend with SQLite/MySQL support

---

# how to run:
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- python3 app.py
